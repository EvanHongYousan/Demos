• TOP1 辅导过程中无法听见或听不清声音问题？
    • 首先双方都确认声音和听筒音量是否打开。有两种情况，一种是老师听不到学生说话，可能是老师手机的声音权限未打开，或学生端听筒音量未打开，请到【手机设置】中允许语音通话的各项限制（最简单的方法，卸载重新安装一次，然后全部点“允许”），提醒学生将音量调大；另一种情况是学生听不到老师说话，可以提示学生使用带话筒的耳机进行辅导。
  • TOP2 老师端显示的辅导累计时长问题？
    • 默认显示为每天辅导的累计时长，所以隔天登录默认时长为0。老师可在辅导时长右上角切换查看今日、本月、累计的时长。
  • TOP3 老师个人资料信息是否保密？
    • 学生端只能查看到老师的头像、性别和姓氏，其它只保存在后台不对外公布。老师的全名、教师资格证、身份证，是内部资料，只能内部系统才能查看，不会泄露老师们的信息。
  • TOP4 只需要有教师资格证的老师吗？大学生可以吗？
    • 我们需要有教学经验的一线老师，至少是教过一轮的老师。
    • 原因有三：其一，能够用最适合学生目前水准理解的解题思路或者解题方法帮助学生； 其二，认知心理学和教育学，能够系统的了解学生身体、心理成长的变化；其三，熟悉自己所教学科一轮的知识考点，敏锐把握各大考点......所以，不是有教师资格证的老师就可以的哦~~我们是一群很认真在做教育的人。
  • TOP5 学生呼叫老师进行骚扰、侮辱，怎么办？
    • 如有学生对老师进行骚扰、侮辱、聊天、不肯挂断等情况（跨学科问问题不算哦）。老师们可以在辅导结束后，点击右上角的举报熊孩子功能。如果学生不愿意挂断，老师要耐心提醒后，方可挂断。
  • TOP6 长时间未审核通过，怎么办？
    • 老师提交资料后，一般在1-2个工作日会审核，如果遇到假期，则会顺延至下个工作日进行审核。如果教师申请的数量比较大，或者账号曾因违规操作被取消资格，老师可先加入QQ群了解详情，QQ群：122237012。
  • TOP7 老师是否能查看到个人好评差评？
    • 老师目前仅能查看到自身的好评数，无法查看到差评数。好评率的计算老师可根据辅导次数与好评数进行评估。我们不会对老师提供差评数。如确实有好评率过低的情况，官方会直接联系相关老师私下沟通。
  • TOP8 老师可以回答非所教学科或学段问题吗？
    • 任何学生呼叫都需要接听，接听后引导学生去呼叫相对应的学科/学段老师（平台禁止辅导跨学科/学段辅导）。 例子口径：同学，这是初中题目，我是小学老师，不擅长讲初中问题哦，建议你在列表中重新找一个初中老师问。
  • TOP9 辅导断线怎么办？
    • 辅导中因为双方的网络或者设备问题，有可能会断线。请老师注意断线后的提示：对方掉线还是自己掉线，还是因为其他问题。如果提示对方中断，属于对方外网络不稳定（或退出软件）导致。如提示自己网络中断与自身网络环境有关，请调整好网络后在尝试辅导，尽量避免不稳定的3G/4G网络或网速较低的局域网下辅导。
  • TOP10 平台上接到呼叫说是官方人员，要求交押金或者上交老师资料信息等情况，怎么办？
    • 官方不会以任何形式对老师征收押金或者要求老师给钱的情况，如果老师遇到此类情况，请在个人中心反馈，反馈时注意写清楚对方的名字。同时，官方也不会在平台上让老师提交任何资料或问任何信息，如果遇到，同上操作。
  • TOP11 高中老师可以辅导初中或者小学吗？
    • 抱歉，老师，我们是一个非常专业的平台。高中老师只能辅导在校所教的高中科目；初中老师也只能辅导在校所教的初中科目；小学老师同样也只能辅导在校所教的小学科目。如果您在学校同时教授两个学科，请您挑选自己最擅长的一科申请答疑。
  • TOP12 不管什么问题，只要学生问，老师都可以回答吗？
    • 不管什么问题，老师都要接听；但老师只能回答跟自己本学段和学科相关的学习类的问题。而且老师在解答问题时候，要用最简单、最省时间、最容易被学生理解的方法，最重要的一点：授之以鱼不如授之以渔（方法永远比答案重要）。
  • TOP13 在平台上，怎么样的辅导方式是最佳的？
    • 建议老师在用语音讲解的同时，辅以关键部分的板书。要在答疑的过程中，多跟学生互动，引发学生思考。老师只是引路人，学生才是学习的主导。切忌学生传题目后，老师拿着题目从头讲到尾，给出老师的做题思路的展示和答案。
  • TOP14 如果老师辅导过程中遇到不会做的题目，该怎么办？
    • 老师不要着急，遇到不会做的题目是很正常的。老师需要以最快的速度确认这道题对于是自己来说比较困难的，然后跟学生商量好。 例子口径：“这题最后一问老师需要验算两种方法，为了节约你的学时，咱们先挂断，你过10分钟后，再呼叫老师，好吗？”
  • TOP15 如果老师辅导过程中遇到不肯开口说话的学生，该怎么办？
    • 这时候，请老师耐心引导学生第一遍“能听到老师说话吗？“； 如果学生还不肯说，引导第二遍”老师只是为了测试咱俩的辅导是否是有效连通的，能听到的话，打个勾好吗？“； 如果还没有反应，那么老师最后提醒“为了不浪费你的学时，老师挂断啦，欢迎使用呼叫老师，再见”。
  • TOP16 学时问题
    • 1. 呼叫老师的学时是什么？
    • 在“呼叫老师”平台上，需要有学时才能呼叫老师进行答疑，新用户都会有20分钟的免费学时，使用完毕后则需要自行充值。
    • 2. 学时卡学段区分
    • 目前区分小学、初中和高中三个学段，如果只充值小学的分钟卡、包月卡的话，仅支持呼叫小学老师，无法呼叫初中老师和高中老师。
    • 3. 充值错误了学段怎么办？
    • 充错学段是无法转移到另一个学段中，包括充错帐号。所以请在充值的时候仔细查看提示避免误操作带来的损失。
    • 4. 提示学时卡余额不足
    • 如未进行过充值，则说明该帐号内免费学时不足。如再想呼叫老师请在个人中心中进行充值。 例子口径：同学，你的学时余额不多了，咱们快速讲完后，你需要在【个人中心】购买学时才能继续使用了。
    • 5. 呼叫老师中使用的学时会扣除话费吗？
    • 不会扣除话费。呼叫老师通话中使用的是流量数据。不扣除任何用户的话费。在购买学时卡时也不会扣除电话费。更不会通过各种形式扣除用户话费。
  • TOP17 有关官方质检问题
    • 1. 质检不合格
    • ①教师在辅导过程中出现质疑、威胁、辱骂等学生的行为或者言语；
    • ②教师把公司对老师的要求，转嫁给学生“公司不允许老师......”；
    • ③教师不按照公司的要求进行辅导；
    • ④教师答疑过程中出现知识点讲解错误；
    • ⑤教师答疑过程中做与学习无关的事情；
    • ⑥教师答疑过程中，故意拖延学生的答疑学时；
    • ⑦教师答疑过程中，只把题目给学生从头到尾计算出来，无方法技巧的引导或沟通。
    • 2. 空白辅导
    • “空白辅导”指的是学生会做的不会做的题目，都一股脑不做，全让老师帮忙讲解。遇到这种情况，老师引导学生先完成1-2题，剩下的让学生自己完成；或者让学生先完成会做的，不会做的再接着问老师。 例子口径：同学，你看，老师带你完成的这两题其实也没你想象中那么难，试试下面的自己完成，不会的老师等会帮你解决。但学习只能自己完成，才会有收获，相信自己可以的......
    • 3. 故意拖延时间
    • 答疑的原则是教师需要以最省时、最简单、最容易被学生理解的方法解答学生上传的题目。同一个知识点，在学生无要求的前提下，不能重复多次讲解；不能使用任何方法或手段去延长学生的辅导时间。
  • TOP18 有关拍照的各种问题
    • 1. 如何拍照上传：
    • 可在呼叫前提前用手机的拍照功能拍下问题，待呼叫成功后直接上传照片。或者在辅导中在屏幕左侧找到相机键进行拍照上传。
    • 2. 多张照片上传：
    • 呼叫老师可支持一次上传20张照片。学生与老师共享，如果学生已经上传10张，则老师只能再上传10张。照片会按照上传顺序排序，展示在20屏中。
    • 3. 老师端拍照上传：
    • 老师可将较长的辅导内容拍照上传给学生。可传多张照片，但学生与老师上传的照片一共不能超过20张。
    • 4. 学生传照片时辅导中断：
    • 先确认是否每次辅导中传照片都会导致辅导中断，如果确认必现可在个人中心反馈中反馈。 有可能是因为手机本身存在的内存不足，影响照片上传中断，建议尝试在使用呼叫老师之前，清理一下内存或者重启一下手机。如果确认之后，还出现问题可在个人中心进行反馈。
    • 5. 无法拍照上传
    • 部分型号手机可能会出现不兼容情况，可在呼叫前先把照片拍下保存在手机中，待呼叫连接成功后选择直接上传照片。 有可能是因为手机本身存在的内存不足，影响照片上传中断，建议尝试在使用呼叫老师之前，清理一下内存或者重启一下手机。如果确认之后，还出现问题可在个人中心进行反馈。
    • 6. 照片上传延迟或失败
    • 由于辅导双方网络环境有差异，会出现失败或延迟的情况，延迟会有几秒-10几秒的等待时间，失败可在辅导中多尝试上传。如尝试多次都无法上传可在个人中心反馈中反馈。
    • 7. 上传照片模糊
    • 手机像素过低，光线较暗、拍照时颤抖都会导致照片模糊，并非产品问题。请更换手机拍照后在上传。
    • 8. 上传的照片是否可以调整转向
    • 用手机自带的相机把题目拍下来，然后保存在手机相册里，调整好转向。然后在辅导中直接“选取照片”，上传给答疑老师，就可以了。在辅导中直接拍照可以做节选部分上传。
  • 温馨提示：如何在平台上，做一个优秀的答疑老师？
    • 第一，老师们要把自己定位在“学习伙伴”的角色上；第二，平台上主要通过语音沟通，所以需要老师们能够让自己的言语，更切合学生的需求。换句话说，就是要有共同语言；第三，答疑过程，要用最简单的方法和技巧，解决学生疑问，体现老师专业性；第四，认识到课堂和平台不一样的地方，以及不断提高自身能力。
